다음 데이터 스키마에 맞는 샘플 데이터를 생성해줘.

CREATE TABLE wagging (
	id	BIGINT	NOT NULL,
	wagger	BIGINT	NOT NULL,
	waggee	BIGINT	NOT NULL

    FOREIGN KEY wagger REFERENCES participant
    FOREIGN KEY waggee REFERENCES participant
);

wagging은 좋아요를 누르는 것과 같은 기능이야.
wagger는 좋아요를 누른 주체를 의미하고, waggee는 좋아요의 대상을 의미해.
wagger와 waggee 모두 외래키이며, 정수 값을 가져.
wagger와 waggee가 참조하고 있는 테이블(participant)의 PK가 1~33까지 있어.
모든 participant 개체가 3~6명 중 랜덤한 개수 만큼의 사람에게 wagging을 한 임의의 데이터를 생성해줘.
단, 한 사람이 같은 사람에게 두 번 wagging 할 수는 없어.

JSON 형식으로 출력해줘.

출력 예시:
[
    {
        "id": 1,
        "wagger": 1,
        "waggee": 33,
    },
    {
        "id": 2,
        "wagger": 1,
        "waggee": 7,
    }
]
