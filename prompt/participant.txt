task 설명:
당신은 임의의 사람 데이터를 생성해야합니다. 생성하는 사람은 대한민국의 대학생입니다. 
그리고 각각의 학생들은 다음과 같은 정보를 가지고 있어.
- part: 본인이 팀에서 맡을 역할이며 ["pm", "de", "fe", "be"] 중 하나입니다.
- team_vibe: 선호하는 팀의 분위기입니다. ["learning", "professional"] 중 하나입니다.
- active_hours: 프로젝트에 주로 투자하는 시간대입니다. ["day", "night"] 중 하나입니다.
- meeting_preference: 선호하는 회의 방식입니다. ["online", "offline"] 중 하나입니다.
- openness: 성격 요소 중 개방성을 나타냅니다. 새로운 경험, 아이디어, 예술에 대한 관심과 호기심의 정도를 나타냅니다. 0 ~ 1 사이의 소수점 두 번째 자리까지 저장합니다.
- conscientiousness: 성격 요소 중 성실성을 나타냅니다. 책임감, 성취욕, 신뢰성, 계획성 등 목표 지향적인 행동과 관련된 특성입니다. 0 ~ 1 사이의 소수점 두 번째 자리까지 저장합니다.
- extraversion: 성격 요소 중 외향성을 나타냅니다. 사교성, 활동성, 친밀감, 주도성 등 외부 세계와 교류하고 에너지를 얻는 정도를 의미합니다. 0 ~ 1 사이의 소수점 두 번째 자리까지 저장합니다.
- agreeableness: 성격 요소 중 친화성을 나타냅니다. 타인에 대한 배려, 협력, 동정심, 성격의 온화함 등 사회적 조화와 관련된 특성입니다. 0 ~ 1 사이의 소수점 두 번째 자리까지 저장합니다.
- neuroticism: 성격 요소 중 신경성을 나타냅니다. 불안, 우울, 스트레스 민감성 등 정서적 안정성과는 반대되는 부정적인 감정을 느끼는 정도를 나타냅니다. 0 ~ 1 사이의 소수점 두 번째 자리까지 저장합니다.
총 40개의 학생 데이터를 생성해주세요.
파트의 비율은 pm : de : fe : be = 0 : 1 : 2 : 2 정도가 되게 해주세요.
응답은 json 형식으로 출력해야합니다.

응답 예시:

[
    {
        "id": 1,
        "part": "",
        "team_vibe": "learning",
        "active_hours": "day",
        "meeting_preference": "offline",
        "openness": 0,
        "conscientiousness": 0,
        "extraversion": 0,
        "neuroticism": 0,
        "agreeableness": 0
    },
    {
        // 학생 데이터 구조는 모두 동일
    }
]
